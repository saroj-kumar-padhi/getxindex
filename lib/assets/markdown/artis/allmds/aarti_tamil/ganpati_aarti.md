# ஆர்த்தி ஸ்ரீ கணபதி ஜி

கணபதியின் சேவை மங்களகரமானது, சேவையால் அனைத்து தடைகளும் தவிர்க்கப்படும்.

மூன்று உலகங்களின் மொத்த தெய்வங்கள், எப்போதும் வாசலில் நின்று பிரார்த்தனை செய்கின்றன.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

ரித்தி-சித்தி தெற்கு இடது பக்கத்தில் அமர்ந்து, பின்னர் மகிழ்ச்சியுடன் பிரகாசிக்கட்டும்.

ஆரத்தி பக்தர்கள் தூப-தீபத்திற்காக நின்று ஆரவாரம் செய்கிறார்கள்.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

எலி வாகனத்திற்கு வெல்லம் மோதங்கள்.

சாந்தமான வடிவத்தைக் கண்டு கணபதியின் தடைகள் ஓடிவிட வேண்டும்.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

பாதோ மாத ஆரு சுக்ல சதுர்த்தி நாள் வெகு தொலைவில் உள்ளது.

சிம்ம ஜென்ம கணபதி பிரபு ஜிதுர்கா உங்கள் மனம் மகிழ்ச்சியில் நிறைந்திருக்கட்டும்.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

அற்புதமான பஜாவை விளையாடுங்கள், இந்திர கதேவ் சகோதரர்களே, அனைவரும் பாடட்டும்.

ஶ்ரீ ஶங்கரின் ஆனந்த் உபஜ்யானம் ஸுன்யோ சப் விக்ன தாரிந்

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

ஆனை உருவாக்கியவர் அமரும் இருக்கை, இந்திரன் அப்சரா நடனம்.

வேத் பிரம்மா ஜியைப் பாருங்கள், தடைகளை அழிப்பவரின் பெயரைப் பற்றிக் கொள்ளுங்கள்.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

ஏகதந்த கஜ்வதன் விநாயக்த்ரியன் வடிவம் அனூப் தாரைன்.

பகதாம்பா போன்ற வலிமையான வயிறு வேண்டும், சந்திரன் சிரிக்கட்டும்.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

ஸ்ரீ சந்திரதேவரின் சாபத்தை உடனே நிறைவேற்ற வேண்டும்.

பதினான்காம் உலகத்தில் கணபதி மூவுலகிலும் ஆட்சி செய்யட்டும்.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

அதிகாலையில் எழுந்து தியானம் செய்யுங்கள்

பூஜை கால் ஆரத்தி கிராமம், யஷ் சத்ர பிரையை எடுத்துக் கொள்ளுங்கள்.

கணபதிக்கு சேவை செய்யும் மங்கள் மேவா...

முதலில் கணபதியை வழிபடுவதன் மூலம் அனைவருக்கும் தடைகள் நீங்கும்.

பக்தர்கள் அனைவரும் கூப்பிய கரங்களுடன் கணபதியைத் துதிக்க வேண்டும்.
